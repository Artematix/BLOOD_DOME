## Adding an Ability

1. Copy template from `.claude/templates/ability.md`
2. Create draft in `drafts/abilities/{id}.md`
3. When approved, move to:
   - Universal: `content/abilities/universal/{id}.md`
   - Exclusive: `content/abilities/exclusive/{id}.md`
4. Add entry to appropriate index
5. Update shop catalog if purchasable
6. Log in `.claude/HISTORY.md`
