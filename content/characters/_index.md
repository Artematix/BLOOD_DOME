## Adding a Character

1. Copy template from `.claude/templates/character.md`
2. Create draft in `drafts/characters/{id}.md`
3. When approved, move to `content/characters/{id}.md`
4. Create asset folder: `assets/characters/{id}/`
5. Add entry to this index
6. Log in `.claude/HISTORY.md`
