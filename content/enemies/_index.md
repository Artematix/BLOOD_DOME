## Adding an Enemy

1. Copy template from `.claude/templates/enemy.md`
2. Create draft in `drafts/enemies/{id}.md`
3. When approved, move to `content/enemies/{id}.md`
4. Create asset folder: `assets/enemies/{id}/`
5. Add entry to this index
6. Log in `.claude/HISTORY.md`
