## Adding an Item

1. Copy template from `.claude/templates/item.md`
2. Create draft in `drafts/items/{id}.md`
3. When approved, move to `content/items/{category}/{id}.md`
4. Create asset: `assets/items/{id}.png`
5. Add entry to category index
6. Update shop catalog if purchasable
7. Log in `.claude/HISTORY.md`
