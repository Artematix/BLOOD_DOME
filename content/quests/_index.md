## Adding a Quest

1. Copy template from `.claude/templates/quest.md`
2. Create draft in `drafts/quests/{id}.md`
3. When approved, move to `content/quests/{id}.md`
4. Add entry to this index
5. Link from character file if character-specific
6. Log in `.claude/HISTORY.md`
