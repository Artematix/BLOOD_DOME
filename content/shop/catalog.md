# Shop Catalog

The central marketplace of Blood Dome. All purchasable items and abilities.

---

## How the Shop Works

- Players can access the shop during designated shop phases
- Buy price listed next to each item
- Sell items back for **50% of buy price** (unless noted otherwise)
- Some items are **character-exclusive** (marked with character ID)
- Stock may be limited for some items

## Updating the Catalog

When adding a new purchasable item or ability:
1. Create the item/ability file first
2. Ensure it has `shop.buy` and `shop.sell` values
3. Add a row to the appropriate section above
4. Link to the item file for full details
