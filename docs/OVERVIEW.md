# Blood Dome Overview

## What is Blood Dome?

Blood Dome is a custom PvPvE "battle to the death" game mode built on D&D 5e (2024 Rules). It's designed as a one-shot experience where players compete in a gladiatorial arena setting.

## Core Gameplay Loop

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│    SPAWN → FIGHT/LOOT → SHOP → UPGRADE → REPEAT        │
│      │                                      │           │
│      └──────────── UNTIL ONE REMAINS ───────┘           │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### Phase Breakdown

1. **Spawn**: Players choose from premade characters (Level 5)
2. **Combat**: Fight enemies (PvE) and other players (PvP)
3. **Collect**: Earn coins from kills, objectives, quests
4. **Shop**: Spend coins on upgrades, items, abilities
5. **Survive**: Handle map events, hazards, enemy waves
6. **Victory**: Last player standing wins

## Key Systems

### Characters
- Premade builds, no custom creation
- Start at **Level 5**
- Each has unique abilities and progression path
- Personal quests for bonus rewards

### Economy
- **Coins**: Primary currency
- Earned through: PvP kills, PvE kills, quests, events
- Spent at: The Shop

### The Shop
- Central hub for upgrades
- Buy: Weapons, armor, consumables, abilities
- Sell: Unwanted items (typically 50% of buy price)
- Some items are character-exclusive

### Progression
- Characters can level up (beyond 5)
- Each level grants predetermined bonuses
- Some abilities unlock at higher levels

### Quests
- **Universal**: Available to all players
- **Character-specific**: Personal objectives
- Provide coin rewards, items, abilities

### Events
- Periodic map-wide occurrences
- Can be beneficial or hazardous
- Add unpredictability and pressure

## Win Condition

**Last player standing wins.**
