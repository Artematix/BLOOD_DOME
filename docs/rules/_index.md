## Adding a Rule

1. Create file in `docs/rules/{rule-name}.md`
2. Add entry to this index
3. Log in `.claude/HISTORY.md`

## Rule Format

```markdown
# Rule Name

## Summary
One-paragraph overview.

## Details
Full rule explanation.

## Examples
Concrete examples of rule in action.

## Notes
Design rationale, edge cases.
```
